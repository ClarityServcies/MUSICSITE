Kontakt Installation Guide

Important: We recommend disabling your anti-virus prior to downloading, otherwise it can remove/corrupt the KeyGen.

1. Mount the provided ISO file (Kontakt 8 8.0.1.iso)

2. Install Kontakt 8 using the provided installer from the mounted ISO (Kontakt 8 Setup PC.exe)

3. Navigate to "%COMMONPROGRAMFILES%\Native Instruments\Service Center"

- If the "Service Center" folder doesn't exist, create it and copy the provided "NativeAccess.xml" file into it

- If there is a "NativeAccess.xml" file inside, check its size. If smaller, replace it with the provided .xml file

4. Activate Kontakt 8 using the provided KeyGen (Kontakt_Keygen.exe)
